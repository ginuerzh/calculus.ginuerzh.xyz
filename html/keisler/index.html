2.8 Implicit Functions We now turn to the topic of implicit differentiation. We say that \(y\) is an implicit function of \(x\) if we are given an equation $$ \sigma(x, y) = \tau(x, y) $$ which determines \(y\) as a function of \(x\). An example is \(x + xy = 2y\). Implicit differentiation is a way of finding the derivative of \(y\) without actually solving for \(y\) as a function of \(x\). [More]
